# Product

## Descrição
O Dashboard de Operações é um SaaS para gestão de operações de tráfego pago (ADS), com foco em controle financeiro por operação e visão consolidada por usuário.

O sistema permite acompanhar:
- custo
- receita
- lucro
- ROI
- repasse e repasse líquido

## Estado Atual
- Login com Supabase Auth funcionando.
- Estrutura multiusuário ativa com isolamento por `user_id`.
- Dashboard principal operacional.
- Página de operação com blindagem por URL (`id` + `user_id`).
- CRUD de operações e despesas em funcionamento.

## Objetivo Final
Transformar o produto em um SaaS robusto e escalável, com:
- segurança forte de dados por usuário
- confiabilidade nos cálculos financeiros
- experiência profissional para análise de performance
- base pronta para monetização (planos e billing)
