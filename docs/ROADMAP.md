# Roadmap

## Segurança
- Implementar RLS nas tabelas principais (`operacoes`, `lancamentos`, `despesas`).
- Garantir validação de ownership em todas queries críticas.

## UX Base
- Adicionar logout explícito na interface.
- Exibir usuário logado no topo da aplicação.

## Produto
- Evoluir dashboard com mais clareza visual e análise temporal.
- Preparar trilha para recursos premium.

## SaaS e Monetização
- Definir planos (ex.: Free/Pro) com limites por usuário.
- Integrar Stripe para assinatura e cobrança recorrente.

## Operação
- Consolidar regras de segurança e contexto em docs para onboarding rápido.
